📋 How to Use

Step 1: Add New District

  1. Enter district name (e.g., "callowhill", "frontier")
  2. Select which CSV files you have (Weekday, Saturday, Sunday)
  3. Upload the CSV files
  4. Click "Add District"

Step 2: Upload CSV Files

  1. Click the upload area to select your CSV file
  2. Only CSV files are accepted
  3. File should match your SEPTA format
  4. Upload one file at a time
💡 Tip: Your CSV files can have any column names. The app will automatically detect and parse them correctly based on content.
